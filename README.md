# Outer_Earth

Outer Earth is an interactive astrology learning tool. Utilizing the NASA API and the ISS API to dynamically update current astrological events.



## BUILT WITH 
* HTML
* CSS 
* JAVASCRIPT/JQUERY
* BOOTSTRAP 
* GSAP Animations
* Scroll magic
* Font Awesome
* Google Fonts

## API's USED
* NASA 
* ISS
